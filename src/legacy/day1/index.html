--- Day 1: Secret Entrance ---

[View Puzzle]

This puzzle involves simulating a circular dial with 10 positions (0-9) that can be rotated clockwise or counterclockwise. The challenge is to track how many times the dial passes through specific positions during a series of rotations.

Part 1: Count Ending Positions at 0

In Part 1, we count how many rotations end with the dial at position 0. Starting from position 0, we apply each rotation instruction and check if the final position equals 0.

Part 2: Count All Clicks Through 0

In Part 2, we count every single click through position 0 during rotations, not just the ending positions. For example, a rotation of +3 from position 8 passes through positions 9, 0, 1, and 2 — that's one click through 0.

Answer: See solution2.py output

Implementation

The solution is implemented in Python using matplotlib for visualizations. The dial is represented as a circular plot with 10 positions, and each rotation is tracked step-by-step.
